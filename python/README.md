# PHC2024 PB Interconnections

This repository is an archive of the script and files used in the PB Interconnections visualization of the Planetary Health Check 2024.

## Data

The XLSX file contains the data that is visualized (with some additional info). It is the result of literature review done by Adrian and Anja.

## Script

The Python script used to generate the plots is found in the Jupyter notebook "PB Interconnections.ipynb". There are multiple parameters to customize the visuals.
